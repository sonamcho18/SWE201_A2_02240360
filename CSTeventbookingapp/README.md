# CST Event Booking App

## App Idea

The CST Event Booking App helps college students book campus facilities easily. Students can browse, view details, and book facilities with a simple swipe.

Available facilities: Turf, Indoor Hall, Convention Hall, Labs, and Teamwork Hall.

---

## Features

The app has five main screens.

**Home Screen** shows a welcome message and quick action buttons. It also displays featured facilities.

**Categories Screen** lists all bookable facilities. Students can see every option in one place.

**Detail Screen** shows full information about a facility. This includes capacity, price, and availability. Students swipe right to book.

**Profile Screen** displays student information. It shows name, student ID, department, and booking statistics.

**Animation Demo Screen** showcases all animations used in the app.

---

## Navigation Flow

The app uses two types of navigation.

**Bottom Tab Navigation** lets users switch between the main screens. The tabs are Home, Categories, Profile, and Animations.

**Stack Navigation** handles screen transitions. When a user taps a facility, the Detail Screen slides in from the right.

The back button returns to the previous screen.

---

## Animations Used

**Fade Animation** appears on the Home Screen header. The text fades in smoothly when the screen loads.

**Scale Animation** happens when pressing facility cards. The card shrinks slightly then springs back to normal size.

**Slide Animation** controls how screens change. New screens slide in from the right side.

**Loading Progress Animation** shows a moving progress bar. It repeats continuously on the Animation Demo Screen.

**Pulse Animation** makes a box grow and shrink. It also repeats on the Animation Demo Screen.

---

## Gesture Interaction

The app has one gesture interaction.

**Swipe to Book** works on the Detail Screen. Users swipe a button to the right to confirm booking. The button moves with the finger during the swipe.

A popup alert confirms when booking is successful.

---

## Screens Overview

**Home Screen** welcomes Sonam Choki and shows featured facilities.

**Categories Screen** displays all five bookable facilities.

**Detail Screen** shows facility info and has the swipe button for booking.

**Profile Screen** shows student details and booking stats.

**Animations Screen** demonstrates all animations live.

---

## Data Structure

Each facility in the app contains specific information.

This includes name, description, capacity, price, and availability.

All prices are in Bhutanese Ngultrum (Nu.).

---

## Color Palette

The app uses five main colors.

**Blue (#3498DB)** is for buttons, icons, and active tabs.

**Navy (#2C3E50)** is for headers and text.

**Off-white (#F8F9FA)** is the main background color.

**Light Gray (#ECF0F1)** is for cards and icon backgrounds.

**Dark Gray (#7F8C8D)** is for secondary text.
